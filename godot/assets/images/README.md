# Assets de Imagens

## Menu Background

Para adicionar uma imagem de fundo ao menu, coloque uma imagem chamada `menu_background.png` nesta pasta.

A imagem será exibida como fundo do menu principal. Se a imagem não existir, o menu usará uma cor sólida como fallback.

**Recomendações:**
- Tamanho recomendado: 1920x1080 ou maior
- Formato: PNG ou JPG
- Tema: Algo relacionado a torres, labirintos ou defesa
