Coloque aqui seus assets (imagens e sons).

Estrutura sugerida:

images/
- hero.png        (32x32 ou 48x48, com transparência)
- enemy.png       (32x32)
- base.png        (32x32)

sounds/
- shoot.wav       (efeito de tiro da flecha)
- hit.wav         (efeito de acerto no inimigo)

Observações:
- Se um arquivo não existir, o jogo usa formas/cores como fallback.
- Você pode usar outros tamanhos; o jogo fará ajuste simples de escala.
