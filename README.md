# Tower Defence (Godot 4)

Este repositório agora contém somente o projeto Godot.

## Como rodar
- Instale Godot 4.2+.
- Abra a pasta `godot/` (arquivo `project.godot`).
- Rode a cena principal (F5). A main scene já está configurada.

## Estrutura
- `godot/project.godot`: configuração do projeto
- `godot/scenes/Main.tscn`: cena principal (HUD, overlay de upgrades)
- `godot/scripts/Game.gd`: lógica do jogo

## Controles
- Clique: atira na posição do mouse
- Fim de wave: escolha 1 benefício e clique em Resume
- Kill All no HUD: botão de teste

## Observação
- A pasta `assets/` pode ser usada para colocar futuros sprites/sons.
